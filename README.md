# SoftGrowTech_HousePricePrediction

## 📌 Description
This project predicts house prices using Linear Regression.

## ⚙️ Technologies
- Python
- Pandas
- Scikit-learn

## 📊 Steps
- Load dataset (California Housing)
- Data preprocessing
- Train-test split
- Model training
- Evaluation (MAE, R2)

## ▶️ Run
pip install -r requirements.txt
python house_price.py
